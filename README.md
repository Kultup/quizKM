# Telegram-бот для навчання персоналу ресторанів

Цей проект представляє собою Telegram-бота для навчання персоналу ресторанів. Бот дозволяє автоматично розсилати щоденні тести, перевіряти відповіді, надавати пояснення, вести рейтингову систему, надавати доступ до бази знань та збирати зворотний зв'язок від співробітників.

## Функціонал

### Для користувачів (співробітників ресторанів)

1. **Автоматична розсилка питань**
   - Щоденна розсилка тестових питань
   - Нагадування про непройдені тести
   - Дедлайн для проходження тестів

2. **Пояснення до відповідей**
   - Детальні пояснення до правильних та неправильних відповідей
   - Посилання на матеріали з бази знань

3. **Рейтингова система**
   - Нарахування балів за правильні відповіді
   - Рейтинг співробітників
   - Статистика проходження тестів

4. **База знань**
   - Доступ до навчальних матеріалів
   - Категоризація матеріалів
   - Пошук по базі знань

5. **Зворотний зв'язок**
   - Можливість надати відгук про роботу бота
   - Пропозиції щодо покращення навчальних матеріалів

### Для адміністраторів

1. **Адміністративна панель**
   - Управління користувачами
   - Управління питаннями та категоріями
   - Управління базою знань
   - Перегляд статистики та звітів
   - Експорт даних

## Технічний стек

- **Мова програмування**: Python 3.8+
- **Фреймворк для Telegram**: python-telegram-bot
- **База даних**: SQLAlchemy з підтримкою PostgreSQL/SQLite
- **Адміністративна панель**: Flask
- **Планувальник завдань**: schedule
- **Обробка даних**: pandas
- **Візуалізація**: matplotlib

## Встановлення та запуск

### Передумови

- Python 3.8 або вище
- pip (менеджер пакетів Python)
- PostgreSQL (опціонально, можна використовувати SQLite)

### Кроки встановлення

1. Клонуйте репозиторій:

```bash
git clone https://github.com/username/restaurant-training-bot.git
cd restaurant-training-bot
```

2. Створіть віртуальне середовище та активуйте його:

```bash
python -m venv venv

# На Windows
venv\Scripts\activate

# На Linux/Mac
source venv/bin/activate
```

3. Встановіть залежності:

```bash
pip install -r requirements.txt
```

4. Створіть файл `.env` з наступними змінними оточення:

```
TELEGRAM_TOKEN=your_telegram_bot_token
DATABASE_URL=postgresql://username:password@localhost/dbname
# або для SQLite
# DATABASE_URL=sqlite:///bot.db
FLASK_SECRET_KEY=your_secret_key
```

5. Ініціалізуйте базу даних:

```bash
python -c "from models import create_tables; create_tables()"
```

### Запуск бота

```bash
python bot.py
```

### Запуск адміністративної панелі

```bash
python admin.py
```

Адміністративна панель буде доступна за адресою http://localhost:5000

Логін: admin
Пароль: admin123

## Структура проекту

- `bot.py` - Головний файл бота
- `handlers.py` - Обробники команд Telegram
- `models.py` - Моделі бази даних
- `database.py` - Функції для роботи з базою даних
- `config.py` - Конфігураційні параметри
- `admin.py` - Адміністративна панель
- `templates/` - Шаблони для адміністративної панелі
- `requirements.txt` - Залежності проекту
